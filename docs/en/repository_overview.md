# Repository Overview: n8n‑nodes‑newrelic

This repository contains the source code for the **New Relic Custom Node** for n8n. It is designed to integrate with New Relic's APIs (specifically the NerdGraph GraphQL API) and extend n8n's capabilities.

## Directory Structure

root
├── docs/                       # Documentation and research notes
│   ├── jp/                    # Japanese documentation
│   └── en/                    # English documentation
├── n8n-nodes-newrelic/         # Main npm package for the n8n node
│   ├── dist/                   # Compiled JavaScript files (generated by build)
│   ├── nodes/                  # Node definitions and logic
│   │   └── NewRelic/           # "New Relic" node implementation
│   │       ├── NewRelic.node.ts # Main logic (input definitions, execution, API calls)
│   │       └── newrelic.svg    # Node icon
│   ├── credentials/            # Credential definitions

│   ├── package.json            # Project settings and dependencies

└── README.md                   # Repository README (simplified version)

## Key Components

### 1. `n8n-nodes-newrelic/nodes/NewRelic/NewRelic.node.ts`
The core of the integration. It defines:
*   **Operations**: What the node can do (e.g., `Execute NRQL`).
*   **Inputs**: Fields displayed to the user (e.g., the NRQL query string).
*   **Execution Logic**: Functions that send requests to `https://api.newrelic.com/graphql`.

### 2. `n8n-nodes-newrelic/credentials/NewRelicApi.credentials.ts`
Defines how user authentication is handled.
*   Requires a **User API Key** (`NRAK‑...`).
*   Includes a **Region** selector (US/EU) to switch between API endpoints.
